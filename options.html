Grayscaler Options

Pin the extension to the toolbar to access settings for the current website, tab, or all websites.

For settings to apply properly, enter the full domain for a website (e.g., www.google.com, not google.com).

Included

Add a website or list of space/comma-separated websites to enable grayscale on.

Not a valid website or list of space/comma-separated websites.

    Excluded

    Add a website or list of space/comma-separated websites to disable grayscale on.

    Not a valid website or list of space/comma-separated websites.


      A website may not have a subdomain (e.g., elijahcantu.com, not www.elijahcantu.com).

      Eventually, you will not need to enter the full domain, unless you want to add a subdomain.

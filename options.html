Grayscale the Web - Manage Saved/Excluded Sites

Use the forms below to manually enter sites you would like saved or excluded.

Make sure to add the site name with the end part after the ".", like a .com or a .gov, otherwise your entry won't work, even if it's appears in the list.

Example: If you wanted to add www.google.com, you would enter "google.com", not "google".

Please note that subdomains won't work with this. So if you add something like "pictures.facebook.com", it will just add "facebook.com".

If all of this is too confusing, it may be easier to add/remove sites using the pop up menu as you go by clicking the Grayscale the Web icon in your chrome menu bar.

Saved Sites:

These sites will always appear gray, regardless of other settings.

Please make sure to add a full website domain with a beginning and end part, like "google.com" rather than just "google".

    Excluded Sites:

    These sites will not turn gray when you have "For This Tab" or "For All Sites" modes on.

    Please make sure to add a full website domain with a beginning and end part, like "google.com" rather than just "google".

      Remove All Background Colors and Images No Matter What:

      Some websites use background images and colors in a way that can't be grayscaled, so you will still see some color there. If you'd like, you can use this toggle to make it so that they will be always removed and replaced with a light gray background.

      Note that if you toggle a site back to color with this setting on, you'll need to refresh the page for the background to go back to normal.
